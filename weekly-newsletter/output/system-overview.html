Technologies

Tool / LibraryCategoryRole in the system
Python 3 Language All generators, data fetchers, site builders, and email scripts
yfinance Data Fetches live OHLCV price data for indices, FX pairs, and futures from Yahoo Finance
pandas / pandas_datareader Data Data transformation, % change calculations, FRED cross-verification
Jinja2 Templating Renders newsletter Markdown from structured data context
markdown (Python lib) Templating Converts newsletter Markdown to HTML for email delivery
weasyprint Rendering Generates PDF exports from HTML (newsletter + daybreak archives)
matplotlib Visualisation Generates price chart PNGs (5-day OHLC bar chart) embedded in newsletters
smtplib (stdlib) Email Gmail SMTP with STARTTLS for sending newsletters and welcome emails
requests / feedparser Network RSS feed fetching (Reuters, AP, CNBC) for Daybreak market news section
python-dotenv Config Loads API keys and Gmail credentials from config/api_keys.env
Static HTML / CSS Frontend All site pages — no JS framework, no build step. CSS Grid + Flexbox, inline styles for email
Git / GitHub Infra Version control + deployment trigger. Push to master = site live in ~60 seconds
