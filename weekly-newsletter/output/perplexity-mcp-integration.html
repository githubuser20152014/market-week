Step-by-Step Process
STEP 0 · NEW
Perplexity Price Fetch
3 queries run in parallel via mcp__perplexity__perplexity_ask with search_recency_filter: "day"
STEP 0 · NEW
Bounds Check & Fixture Build
Sanity check each price. Call build_perplexity_fixture.py to save fixture with prev_close + daily_pct computed automatically.
STEP 1
Generate Markdown
publish_daybreak.sh detects fixture → runs generator without --live. Displays .md for review.
STEP 2
Content Review
User reviews full newsletter. Edits applied directly to .md if needed before publishing.
STEP 3
Publish
--publish: social posts, site rebuild, git commit + push → GitHub Pages live.
STEPS 4–5
Email Review & Send
Preview HTML generated. User confirms → send_email.py dispatches to subscribers.
